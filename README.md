# 🧮 Calculadora React

Projeto desenvolvido com React, inspirado na experiência de uso da calculadora do iPhone. O objetivo foi aplicar conceitos fundamentais do React na construção de uma aplicação interativa, explorando componentização, gerenciamento de estado, renderização dinâmica e persistência de dados.

## 🧠 Sobre o projeto

A aplicação permite realizar operações matemáticas por meio de uma interface moderna e intuitiva.

Além dos cálculos, o projeto conta com um histórico de operações, que permanece disponível mesmo após atualizar a página, graças à integração com o Local Storage.

## 🧠 O que foi trabalhado

Durante o desenvolvimento, foram aplicados conceitos como:

- ⚛️ Componentização com React
- 📦 Utilização de Props para comunicação entre componentes
- 🔁 Renderização dinâmica com o método map
- 🧠 Gerenciamento de estado com Hooks
- 🌐 Compartilhamento de dados utilizando Context API
- 💾 Persistência de dados com Local Storage
- ⚙️ Utilização do useEffect para recuperação de dados salvos
- 🧮 Implementação da lógica de cálculos
- 📜 Histórico dinâmico de operações realizadas
- 🎨 Estilização utilizando Tailwind CSS

## 🚀 Tecnologias utilizadas
- React
- JavaScript (ES6+)
- HTML5
- JSX
- Tailwind CSS
  
## ✨ Funcionalidades
- ➕ Operações matemáticas
- 🖱️ Interface inspirada na calculadora do iPhone
- 📜 Histórico de operações realizadas
- 💾 Salvamento automático do histórico no navegador
- 🔄 Recuperação dos dados após atualização da página
- 📱 Interface responsiva

## 🎯 Objetivo do projeto

Este projeto tem caráter educacional, com foco em:

- Consolidar os fundamentos do React
- Praticar componentização e reutilização de código
- Trabalhar gerenciamento de estado e Context API
- Aplicar persistência de dados com Local Storage
- Desenvolver aplicações front-end mais interativas e organizadas
